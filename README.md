## Requirements

* TYPO3 >= 11.5 & PHP 8.1+

## Installation

### Composer

```bash
composer require konradmichalik/typo3-environment-indicator
```

### TER

Download the zip file from [TYPO3 extension repository (TER)](https://extensions.typo3.org/extension/typo3_environment_indicator).
